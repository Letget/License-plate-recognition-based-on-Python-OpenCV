# License-plate-recognition-based-on-Python-OpenCV

## 功能
1、静态车辆图像车牌识别功能
2、视频车辆车牌识别功能
3、实时车辆车牌识别功能（仅在笔记本自带摄像头执行验证）

## 版本
v1: 原始版本。
v2: 待优化。
